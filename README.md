# Celula-eucariota-
 Este modelo representa una célula animal creada en HTML, mostrando un corte que permite observar sus organelos internos. Incluye núcleo, membrana, citoplasma, retículo, Golgi, mitocondrias y citoesqueleto. Es interactivo, permitiendo seleccionar partes para conocer sus funciones y facilitar el aprendizaje.
